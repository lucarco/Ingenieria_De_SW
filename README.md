Ingeniería de Software 2

Proyecto en Java usando distintas estructuras. 

**Sistema de Gestión Integral (Casino y Parking) | Java, Arquitectura N-Capas, Patrones de Diseño**
- Desarrollo de una aplicación empresarial en Java para la gestión de usuarios, mesas de juego y reservas de parking, implementando una arquitectura estricta en N-capas (Vista, Control, Servicio de Aplicación y Acceso a Datos). 
- Aplicación intensiva de patrones de diseño arquitectónicos, destacando el uso de Data Access Object (DAO) y Facade (Fachada) para abstraer y desacoplar la lógica de negocio de la persistencia de datos (módulos BBDD). 
- Modelado avanzado orientado a objetos para administrar el dominio del sistema, manejando jerarquías y roles de acceso (Clientes frente a Trabajadores), transacciones de juegos y control de ocupación de plazas. 
- Construcción de una interfaz gráfica modular mediante paneles interactivos independientes (MainWindow, AltaJuegoPanel, OcupaPlazaPanel, ApostarPanel) integrados de forma fluida bajo el paradigma Modelo-Vista-Controlador (MVC).
